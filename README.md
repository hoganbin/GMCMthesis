## 2021 华为杯数学建模竞赛 LaTeX 论文模板

1. 正式竞赛时间：**2021年10月14日8:00至10月18日12:00** 

2. 论文模板沿用[2020 研赛](https://github.com/latexstudio/GMCMthesis)以及[2021格式版](https://github.com/andy123t/GMCMthesis)，今年模板并没有格式变化，就是封面换成了华南理工大学，而我在做了一个地方改进，便是对附录的代码块环境做个修改，源码改动在`gmcmthesis.cls`中的第144-251行，实现效果如`example.pdf`的附录.
